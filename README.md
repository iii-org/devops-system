# To run the API server
1. Install python3.6+ and pip on your machine.
2. `$ cd build/devops_api`
3. `$ pip install -r requirements.txt`
4. `$ cd apis`
5. `$ py api.py` # Replace `py` command with your python3 executable on your platform
